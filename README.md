# Página Pessoal - Gustavo Henrique

Projeto desenvolvido para a disciplina **GAC116 - Programação Web (UFLA)**.

Este projeto consiste na criação de uma página pessoal utilizando **HTML, CSS e JavaScript**, com duas versões distintas utilizando diferentes frameworks CSS, conforme solicitado na atividade.

---

https://gustavohmfilho.github.io/pagina-pessoal/bootstrap/

🔹 **Versão com Tailwind CSS**
https://gustavohmfilho.github.io/pagina-pessoal/tailwind/

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap
* Tailwind CSS

---

## 📱 Funcionalidades

* Layout responsivo (adaptável a diferentes dispositivos)
* Tema claro e escuro com alternância (toggle)
* Estrutura organizada de arquivos
* Informações pessoais, projetos e contato

---

## 📌 Objetivo

Aplicar os conceitos de desenvolvimento web abordados na disciplina, incluindo:

* Estruturação de páginas
* Estilização com frameworks
* Responsividade
* Organização de código
